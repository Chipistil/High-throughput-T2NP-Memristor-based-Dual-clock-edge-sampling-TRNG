🚀 Source Code and Partial Datasets

This repository hosts the source code and partial experimental datasets used in our study. The code includes the implementation of delay and relaxation time conversion, waveform-to-bitstream conversion, and verifications for dual-image encryption. The datasets include NIST test results of TRNGs under different parameters.
