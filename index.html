🚀 Source Code and Comparative NIST Test Results

This repository hosts the source code and partial experimental datasets used in our study. The code includes the implementation of delay and relaxation time conversion, waveform-to-bitstream conversion, and verifications for dual-image encryption. The comparative NIST test results include NIST SP800-22 statistical test results of TRNGs under different parameters.
